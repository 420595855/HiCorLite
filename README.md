# HiCorLite

轻量级医院信息系统（HIS）— Hospital Information System

## 项目简介

HiCorLite 是一套面向中小型医院的 HIS 系统，覆盖门急诊、住院、电子病历、药品管理、收费财务、医技、医保对接等核心业务。

## 技术栈

| 层级 | 技术 |
|---|---|
| 后端 | Spring Boot 3 + Spring Cloud + MyBatis-Plus |
| 前端 | Vue 3 + Element Plus |
| 移动端 | Flutter |
| 数据库 | MySQL 8.0 + Redis + Elasticsearch |
| 消息队列 | RocketMQ |
| 部署 | Docker + Kubernetes |

## 模块概览

| 模块 | 说明 |
|---|---|
| 基础平台 | 组织架构、权限、多租户、字典、工作流、日志 |
| 门急诊 | 预约挂号、分诊、医生站、护士站、急诊、自助 |
| 住院 | 入院、医生站、护士站、手术、会诊、出院 |
| 电子病历 | 病历编辑器、质控、模板、签名 |
| 药品管理 | 药库、采购、门诊/住院药房、合理用药 |
| 收费财务 | 门诊收费、住院收费、票据、医保结算 |
| 医技 | LIS检验、PACS影像、病理、超声、心电、血库 |
| 医保 | 目录对照、实时结算、DRG/DIP |
| 患者服务 | EMPI、互联网医院、随访 |
| 运营决策 | 院长驾驶舱、科室分析、绩效 |
| 质控安全 | 临床路径、抗菌药物、不良事件 |
| 供应链 | 耗材、设备管理 |
| 人力资源 | 排班、执业资质 |
| 集成平台 | ESB消息总线、协议适配 |
